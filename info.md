# Family Mapper AI

Real-time family tracking for Home Assistant

## Features
- 🗺️ Live family tracking
- 🚗 Automatic trip detection  
- 📍 Zone monitoring
- 🔔 Smart notifications
- 📊 Analytics dashboard

## Installation
After installing through HACS, add to your dashboard:
```yaml
type: iframe
url: /hacsfiles/HACS-AI-Family-Tracker/index.html
aspect_ratio: 100%
```
